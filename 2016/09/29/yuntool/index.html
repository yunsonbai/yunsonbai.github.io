项目简介

  • 应用：运营数据统计；快熟查询数据库异常数据；快速数据进行增删改查
  • 版本：0.4
  • 安装：
    • git clone git@github.com:yunsonbai/yuntool.git
    • python setup.py install
  • 主要功能和特点:
    • 采用orm方式操作数据库
    • 支持画图
      • 曲线图
      • 柱形图
    • 制表excel
    • 支持邮件发送
      • 纯文字
      • 图加文字
